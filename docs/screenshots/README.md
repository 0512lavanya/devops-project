# Screenshots

Add these images for your submission or portfolio (PNG recommended, 1280px wide).

| File | What to capture |
|------|-----------------|
| `home.png` | http://localhost:5000 home page |
| `login.png` | Login form after successful demo login |
| `health.png` | `/health` JSON response |
| `github-actions.png` | Green CI/CD workflow run on GitHub |
| `dockerhub.png` | Image tags on Docker Hub |
| `deployment.png` | Live URL (Render or EC2) |

### Quick capture (Windows)

```powershell
# Run app first, then use Snipping Tool or:
Start-Process "http://localhost:5000"
```

Reference them in the main README once saved here.
